# 32. Introducción

El presente proyecto describe el análisis, diseño y desarrollo de un sistema web de comercio electrónico orientado a la venta de aparatos electrónicos. La plataforma permite a los clientes explorar un catálogo digital, gestionar direcciones, realizar compras en línea y dar seguimiento a sus pedidos en tiempo real.

El sistema se desarrolla bajo arquitectura MVC utilizando ASP.NET Core, lenguaje C# y base de datos MySQL, garantizando escalabilidad, seguridad y rendimiento.

---

# 33. Objetivos

## 33.1 Objetivo General

Desarrollar una aplicación web e‑commerce para la venta de productos electrónicos que optimice el proceso de compra y gestión administrativa.

## 33.2 Objetivos Específicos

* Implementar catálogo digital interactivo.
* Desarrollar carrito y checkout.
* Integrar pagos en línea.
* Gestionar inventario.
* Implementar notificaciones.
* Permitir historial de direcciones.
* Desarrollar panel administrativo.

---

# 34. Justificación

El crecimiento del comercio electrónico exige plataformas digitales eficientes que permitan a las empresas comercializar productos tecnológicos de forma segura y escalable.

Este sistema busca:

* Digitalizar ventas.
* Reducir errores manuales.
* Mejorar experiencia del cliente.
* Automatizar inventario.
* Incrementar alcance de mercado.

---

# 35. Alcance

El sistema contempla:

* Venta de electrónicos.
* Gestión de usuarios.
* Direcciones múltiples.
* Pagos online.
* Envíos.
* Notificaciones.
* Panel admin.

No contempla:

* Marketplace externo.
* Integración contable avanzada.
* App móvil nativa (fase futura).

---

# 36. Conclusiones

El desarrollo del sistema demuestra la aplicación práctica de tecnologías modernas como ASP.NET Core, C# y MySQL en la construcción de soluciones e‑commerce escalables.

La implementación de módulos como historial de direcciones, notificaciones en tiempo real, gestión de imágenes y panel administrativo permite ofrecer una experiencia completa tanto para clientes como para administradores.

El proyecto es viable para implementación real y escalable hacia nuevas funcionalidades como inteligencia artificial recomendadora o aplicaciones móviles.
